## Описание проекта

**Дипломный проект:** 
Система CAPTCHA на основе анализа траекторий взгляда с использованием eye-tracking

Для понимания идеи разработки проекта , нужно уверенно понимать следующие основные моменты:
 - разобрать ядро алгоритма(файл scorer.py), а именно работу каждого признака,особенно latency!
 - разобрать генерацию искусственной траектории человеческого взгляда(не просто) файл human_simulator.py
    - линейная интерполяция(x = start_x + α * (target_x - start_x) + noise)
    - экспоненциальное затухание(нелинейность) - alpha = 1 - math.exp(-3 * alpha_linear)
    - отрисовка графика траектории
    - анализ таректории velocity
